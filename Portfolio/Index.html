Overall Reflection

Before enrolling in the stack summer camp, the only coding experience I had was with java in the eclipse workspace. I’ll admit it was frustrating trying to learn a new concept that others already seemed familiar with. However, the frustration proved rewarding as it taught me how to independently learn and ask others for help. When faced with a dilemma I didn't know how to solve, I would first independently look for a solution using my resources (the google drive, notes I took, stack overflow, etc.) and then reach out to others if I could not come to a conclusion. This taught me the essential skill of independent learning which will be very useful in college and my career!
